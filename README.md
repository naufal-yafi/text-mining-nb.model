2. Python
3. Pip

## How to use this project

1. Create VENV

```sh
sh run -cv
```

2. Activate VENV

```sh
source active
```

3. Create Kernel

```sh
sh run -ck
```

4. Install All Libraries

```sh
sh run -i
```

5. Setup file .env

```sh
cp .env.example .env
```

Add field `BASE_DIR=`

## Export model to PKL

```sh
sh run -e
```

## Running App

1. Export model to PKL

```sh
sh run -e
```

2. Run server app

```sh
sh run -s
```
