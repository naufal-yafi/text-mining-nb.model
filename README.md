## Software Required

1. Jupyter Notebook
2. Python
3. Pip

## How to use

1. Create VENV

```sh
sh run -cv
```

2. Activate VENV

```sh
source active
```

3. Create Kernel

```sh
sh run -ck
```

4. Install All Libraries

```sh
sh run -i
```

## Export model to PKL

```sh
sh run -e
```
